Nesse estamos desenvolvendo basicamente um cadastro de filmes da aplicação Screen Match, em que precisaremos implementar o formulário da imagem acima. Para isso, utilizaremos tecnologias web como o HTML integrado ao código em Java.

No formulário, preencheremos algumas informações do filme, como o nome, a duração, o ano de lançamento, e o gênero.

Além disso, teremos um botão para concluir o cadastro. Nesse momento, vamos pegar as informações digitadas pela pessoa usuária no navegador, acessá-las no código Java de alguma maneira, e salvar em um banco de dados.

Também trabalharemos em outra página, onde iremos listar os filmes cadastrados no banco de dados:

<p align="center">
  <img alt="MedVoll" src="https://github.com/Julianamcs/screenmatch/blob/main/src/main/java/br/com/jmcs/screenmatch/img/listagem.png" width="100%">
</p>
Esse é o objetivo do curso: transformar a aplicação do Screen Match em uma aplicação web. Dessa forma, conseguiremos acessá-la pelo navegador e também acessar um banco de dados para armazenar as informações.
